# Smart Interview App

AI-powered interview preparation app.

## Tech Stack
- Flutter (Frontend)
- Node.js + Express (Backend)
- Groq LLM API
- MongoDB

## How to Run Backend

cd backend
npm install
create .env file with:
GROQ_API_KEY=your_key
MONGO_URI=your_uri
npm run dev

## How to Run Flutter

cd smart_interview_app
flutter pub get
flutter run
